INC0006818 A problem has occurred. Failed to enrol. - Error p4

Issue for customers trying to set up a new EPP account on eurostar.com. 
 
I have done a tealeaf report to try and find the reason for this error but cannot find any reason or pattern. 
We have 86 examples of this from the last 14 days which can be viewed in Tealeaf. No time or date pattern found 
 
Customers have either set up an engaged account (Eurostar profile on .com for log in details) or just logged in to their engaged account and proceeded to the loyalty page www.eurostar.com/uk-en/account/loyalty and then proceed to www.eurostar.com/uk-en/account/loyalty/join-epp
 
When customers have filled in their details correctly and submitted, the are then returned the error A problem has occurred. Failed to enroll.
