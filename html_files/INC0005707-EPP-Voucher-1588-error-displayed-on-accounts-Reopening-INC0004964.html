INC0005707 EPP: Voucher_1588 error displayed on accounts (Reopening INC0004964)

This is a reopening of INC0004964
 
This error message 'Voucher_1588' appears on a vast majority of EPP accounts which have vouchers attached.
 
We don't agree that the old bug (INC0004964) should have been closed.
 
For example in TeaLeaf in the last 24 hours we return over 400 sessions where users have encountered the Voucher_1588 error and a percentage of these customers mention it when contacting us over the phone/email.
 
 
We need to prevent this error message from displaying on the Account page.
