INC0004662 EFT Train Redemption form - Paris Gare Du Nord is not a valid station? CLOSED FIXED SPRINT 13 2.5.14

On the EFT Train Redemption Form if you start typing 'Paris' in to the outbound or return field it will auto-complete with a list of stations and give you the option of using Paris Gare Du Nord which doesn't display on the Station pop-up list.
 
When a user tries to book using this station they receive the error message : Invalid origin destination combination
 
The auto-complete stations should match the station pop-up list.
